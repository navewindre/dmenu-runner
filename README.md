# dmenu-runner
runner for dmenu that parses xdg entries

usage:

build with zig

```zig build```

put ``zig-out/bin/dmenu_runner`` somewhere in your PATH, e.g.

```
sudo cp zig-out/bin/dmenu_runner /usr/bin/
```

run using ``dmenu_run`` script in the repo
